This project contains implementation for a binary search tree:

Following functions are provided:

	* Inserting a node
	* Displaying the tree 
	  - Inorder    (recursive, iterative)
	  - Preorder   (recursive, iterative)
	  - Postorder  (recursive, iterative)
	  - Level-wise
	* Delete a node
	* Calculating height of the tree
	  - DFS  (depth wise)
	  - BFS  (breadth wise)
